Contenu du projet : 
- chenYl.c/main.c                         → point d'entrée du programme
- chenYl.c/animal.c                      → gestion des animaux
- chenYl.c/gestion_animaux.c      → interactions utilisateur (menu, ajout, adoption, etc.)
- chenYl.h/animal.h                      → définitions des structures
- chenYl.h/gestion_animaux.h     → prototypes des fonctions
- EffectifChenYl.txt                       → fichier de sauvegarde des animaux
- Makefile                                     → compilation automatisée

pour compiler il suffit d’écrire  : “make” 
puis d'appeler l’executable .\chenyl
et vous aurait aussi la commande  :"make clean"

et pour l’utilisation du code tout sera indiqué dans le terminal.


Profitez bien de notre chenyl.
